# Analysis-and-Synthesis-of-Radio-Systems-Design-of-a-Superheterodyne-Receiver
This work is about the design of a superheterodyne receiver; it includes a PDF file with the final report (where the selection of the components is analyzed, including also special analysis for the PLLs and a mathematical analysis for the main metrics of the receiver) and a .zip file containing the datasheets of most components. 
The work was an assignment for the course of Analysis and Synthesis of Radiosystems in the Aristotle University of Thessaloniki by Antonios Dimitriou.
